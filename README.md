# homebrew-tap

Homebrew tap for [CloudNativeBergen](https://github.com/CloudNativeBergen) tools.

## Usage

```sh
brew tap CloudNativeBergen/tap
brew install cnctl
```

## Formulae

| Formula | Description                                |
| ------- | ------------------------------------------ |
| `cnctl` | Organizer CLI for Cloud Native Days Norway |

Formulae are automatically updated on each [cnctl release](https://github.com/CloudNativeBergen/cnctl/releases).
